# Sprint 1 Retrospective

## What Went Well

- Clear user stories and acceptance criteria enabled focused implementation.
- Early test writing helped prevent regressions.
- CI/CD pipeline provided fast feedback and confidence in changes.

## What Can Be Improved (Sprint 2 Focus)
